MedApp: A Comprehensive Medicine Stock Management System<br>
MedApp is a web and mobile-based application designed to streamline medicine stock management for pharmacies. The system consists of three modules: User, Pharmacy, and Admin.

Key Features:

- Pharmacy Module: Allows pharmacists to update their medicine stock levels through a dedicated website.
- User Module: Enables users to search for available medicines near pharmacies via a mobile app.
- Admin Module: Enables administrators to add and manage pharmacies, ensuring efficient stock management.

Technologies Used:

- Python
- Django
- SQLyog
- Java
- Android Studio
